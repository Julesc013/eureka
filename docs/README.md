# Eureka Docs

This directory contains the project vision, architecture notes, operations
guidance, roadmap, standards, decisions, and open questions.

Start with:

- `BOOTSTRAP_STATUS.md` for current maturity
- `ROADMAP.md` for staged direction
- `DECISIONS.md` for accepted decisions
- `OPEN_QUESTIONS.md` for unresolved design questions
- `vision/DOCTRINE.md` for core doctrine
- `architecture/TEMPORAL_OBJECT_RESOLVER.md` for the product model
- `operations/TEST_AND_EVAL_LANES.md` for verification lanes

Docs are governance and orientation material. They should not claim production
readiness unless an accepted future milestone provides evidence for it.
