# Privacy And Shared Evidence

Eureka should distinguish carefully between shareable public evidence and
private user behavior.

## Publicly Shareable By Default

When rights and policy allow, public shareable material may include:

- public source metadata
- content hashes
- extraction manifests
- member listings
- public parsed text
- public compatibility facts with evidence
- reduced-form absence findings

## Partly Shareable Later

Some future resolver memory may be shareable only in constrained or aggregated
form:

- resolution-memory patterns
- anonymous usefulness signals
- anonymous source-quality signals

These remain future policy work rather than current implemented behavior.

## Private By Default

Private-by-default material includes:

- local filesystem paths
- installed applications
- local hardware profiles
- opened files
- exact private search history
- account-linked sources
- user-specific strategy memory

## Hard Rule

Eureka should share public evidence, not private user behavior.

This rule should remain in force even if future shared evidence or resolution
memory services are introduced.
