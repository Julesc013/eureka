# Batch 0 Observation Checklist

Before committing an observed Batch 0 record, verify:

- [ ] Query ID verified against `batch_manifest.json`.
- [ ] Exact query submitted recorded.
- [ ] System ID verified.
- [ ] Date/time recorded.
- [ ] Operator recorded.
- [ ] Browser/tool recorded.
- [ ] Filters or scope recorded.
- [ ] Top results recorded manually.
- [ ] First useful result identified or marked absent.
- [ ] Usefulness scores completed.
- [ ] Failure modes completed.
- [ ] Notes mention uncertainty and staleness.
- [ ] No scraping used.
- [ ] No automated result collection used.
- [ ] No fabricated results included.
- [ ] No private/account-only data included.
- [ ] Sponsored or ad results are not counted as organic unless explicitly
      labeled.
- [ ] Parent item/collection results are distinguished from smallest actionable
      units.
