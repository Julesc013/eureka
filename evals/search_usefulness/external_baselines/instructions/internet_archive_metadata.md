# Internet Archive Metadata Search Manual Observation

This procedure is manual only. Do not scrape Internet Archive, automate result
collection, use crawlers, or call live Internet Archive APIs.

1. Open the Internet Archive website manually.
2. Submit the exact query through the visible metadata/catalog search UI.
3. Record the search URL or visible locator, query text, filters, sort order,
   collection scope, and visible result count if shown.
4. Manually record top results and identify whether each result is an item,
   collection, file-level hit, documentation result, or irrelevant result.
5. Note whether the result is only a parent item or the smallest actionable
   unit for the query.
6. Record first useful result rank and usefulness scores from 0 to 3.
7. Note that metadata search may miss full-text/OCR matches.

This observation is time-sensitive and should not be treated as a stable
Internet Archive baseline outside the recorded context.
