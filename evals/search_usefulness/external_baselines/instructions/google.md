# Google Web Search Manual Observation

This procedure is manual only. Do not scrape Google, automate result
collection, use a crawler, or use an external search API.

1. Open Google in a normal browser session chosen by the operator.
2. Record the exact query text submitted.
3. Record the date/time, operator, browser, and any relevant location,
   account, language, or personalization context.
4. Manually transcribe the top visible organic results. Ads or sponsored
   results must not be counted as organic unless they are explicitly marked in
   the observation notes.
5. Record the first useful organic result rank, if any.
6. Score usefulness from 0 to 3 for object-type fit, smallest actionable unit,
   evidence quality, compatibility clarity, actionability, absence explanation,
   duplicate handling, user-cost reduction, and overall usefulness.
7. Note evidence limitations and staleness concerns.

One observation does not establish stable global Google behavior and must not
be used to claim Eureka beats Google globally.
