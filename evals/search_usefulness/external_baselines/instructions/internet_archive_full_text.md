# Internet Archive Full Text/OCR Manual Observation

This procedure is manual only. Do not scrape Internet Archive, automate result
collection, use crawlers, call live Internet Archive APIs, or add OCR/PDF/image
parsing.

1. Use only visible manual full-text or OCR search surfaces.
2. Record whether full-text/OCR search was used and what collection, item,
   issue, or scope was searched.
3. Record exact query text, filters, visible result count, and result ordering
   where the UI exposes those details.
4. Manually record top text/OCR hits.
5. Distinguish page/article/member-level hits from parent scans or collection
   pages.
6. Record OCR errors, missing page locators, and granularity limitations.
7. Score usefulness from 0 to 3 and record first useful result rank if any.

One manual OCR/full-text observation is not a global Internet Archive full-text
truth claim.
