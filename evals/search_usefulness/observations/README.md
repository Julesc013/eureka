# Manual Baseline Observations

This directory is reserved for human-recorded external baseline observations.

Do not place scraped Google, Internet Archive, or other automated search output
here. Search Usefulness Audit v0 treats external systems as
`pending_manual_observation` unless a human reviewer records and reviews an
observation file.

Manual External Baseline Observation Pack v0 now keeps the governed schema,
instructions, pending slots, validator, and status report under
`../external_baselines/`. Prefer that area for new manual observations.
