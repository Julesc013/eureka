These commands are local development and verification commands. They are not deployment commands.

Clone And Verify

git clone https://github.com/Julesc013/eureka.git
cd eureka
python -m unittest discover -s tests -t .
python scripts/run_archive_resolution_evals.py
python scripts/run_search_usefulness_audit.py
python scripts/public_alpha_smoke.py

Try CLI Queries

python scripts/demo_cli_workbench.py query-plan "Windows 7 apps"
python scripts/demo_cli_workbench.py search "windows 98 registry repair"
python scripts/demo_cli_workbench.py sources
python scripts/demo_cli_workbench.py source internet-archive-recorded-fixtures
python scripts/demo_cli_workbench.py evals-archive-resolution

Try The Local Web/API Workbench

local_dev is for trusted local experiments with local path controls. public_alpha blocks public-alpha unsafe local path controls and local write/readback route groups.

python scripts/demo_web_workbench.py --serve --mode local_dev
python scripts/demo_web_workbench.py --serve --mode public_alpha

Try The Local HTTP API Helper

python scripts/demo_http_api.py --mode public_alpha status
python scripts/demo_http_api.py --mode public_alpha query-plan --query "latest Firefox before XP support ended"
python scripts/demo_http_api.py --mode public_alpha sources
