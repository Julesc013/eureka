# Minimal Local Staging Manifest v0

This directory is a synthetic committed example for Local Staging Manifest Format v0.

It is not runtime staged state, does not create a local quarantine directory, does not copy pack files, and does not affect public search, local indexes, the runtime source registry, snapshots, relay surfaces, uploads, or the master index.

`LOCAL_STAGING_MANIFEST.json` records local/private candidate semantics, validate-only report provenance, staged pack references, staged candidate entities, no-mutation guarantees, and future reset/delete/export policy. The example uses repo-relative synthetic references only and contains no private local paths, credentials, executable payloads, raw databases, or user data.
