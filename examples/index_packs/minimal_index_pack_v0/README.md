# Minimal Index Pack v0

This synthetic example demonstrates Index Pack Contract v0 with public-safe
summary metadata only. It describes a tiny summary-only local-index build over
two synthetic sources and four public-safe record summaries.

The pack is not a raw cache, not a SQLite database, not an importable runtime
index, not an upload, and not a master-index submission. It exists only for
contract and validator testing.

Contents:

- `INDEX_PACK.json` manifest
- `PRIVACY_AND_RIGHTS.md` privacy and rights posture
- `index_summary.json` build and count summary
- `source_coverage.json` source-family coverage summary
- `record_summaries.jsonl` public-safe record summaries
- `field_coverage.json` searchable/display/internal field posture
- `query_examples.jsonl` non-production query examples
- `CHECKSUMS.SHA256` checksums for the pack files

The example contains no executable files, installers, binary artifacts, private
paths, raw payloads, live URLs, or database files.
