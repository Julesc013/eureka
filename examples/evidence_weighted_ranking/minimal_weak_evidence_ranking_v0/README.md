# minimal_weak_evidence_ranking_v0

Synthetic public-safe Evidence-Weighted Ranking Contract v0 example.

This example is contract-only. It does not implement runtime ranking, change public search ordering, suppress or hide results, promote candidates, call live sources, use popularity/telemetry/ad/user-profile signals, or mutate public/local/runtime/master indexes, source cache, evidence ledger, or candidate index.
