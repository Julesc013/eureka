# Minimal AI-Assisted Drafting Flow v0

This directory is a synthetic example for AI-Assisted Evidence Drafting Plan
v0. It is not runtime output. No model call was performed, no provider was
loaded, no private data was processed, and no evidence or contribution was
accepted.

The flow shows a public-safe synthetic input context, a hand-authored typed AI
output example, an evidence candidate, and a contribution candidate. The typed
output must validate before any future drafting tool can use it. The evidence
and contribution examples are review-required candidates only.

The example does not mutate public search, local indexes, runtime source
registry state, staged state, or the master index.
