# Minimal Evidence Pack v0

This synthetic example demonstrates Evidence Pack Contract v0. It contains
public-safe claims about an illustrative browser release, a member path, and an
absence note without including a source fixture, local cache, executable,
installer, private path, live URL fetch, or master-index acceptance claim.

Validate it with:

```bash
python scripts/validate_evidence_pack.py --pack-root examples/evidence_packs/minimal_evidence_pack_v0
```

The evidence records are claims and observations only. They are not canonical
truth, rights clearance, malware safety, or production search relevance.
