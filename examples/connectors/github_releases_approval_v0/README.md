# GitHub Releases Connector Approval Example v0

Synthetic approval-only example for a future GitHub Releases release-metadata connector. It performs no GitHub API calls, no repository clone, no release fetch, no tag fetch, no release asset download, no source archive download, no public-query fanout, and no source cache/evidence ledger/candidate/public/local/master index mutation.

Repository identity is the synthetic placeholder `example-org/example-project`; it is never fetched. User-Agent/contact values, source policy review, token policy, rate limits, timeouts, retry/backoff, and circuit breakers remain pending approval gates.
