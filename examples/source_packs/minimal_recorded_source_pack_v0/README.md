# Minimal Recorded Source Pack v0 Example

This synthetic source pack demonstrates Source Pack Contract v0. It is a tiny
shareable-candidate fixture for validation and documentation tests only.

The pack contains:

- one source record
- two public-safe evidence records
- one representation record
- one tiny synthetic JSON metadata fixture
- rights/access notes
- SHA-256 checksums for all pack files except `CHECKSUMS.SHA256`

This pack is not imported into the runtime, not indexed, not uploaded, and not
accepted by a hosted or master index. It does not grant live network access and
does not contain executable payloads, installers, private local paths,
credentials, raw private files, malware-safety claims, or rights-clearance
claims.
