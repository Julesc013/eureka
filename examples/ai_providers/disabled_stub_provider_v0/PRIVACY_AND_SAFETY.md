# Privacy And Safety

The disabled stub provider is public-safe and synthetic. It contains no API
keys, secrets, credentials, private paths, prompt logs, telemetry payloads,
runtime endpoint configuration, model weights, executables, installers,
downloads, or copyrighted long-form text.

AI Provider Contract v0 keeps all AI providers disabled by default. Future
local providers require explicit local policy, and future remote providers
require explicit credentials, consent, and review before any runtime is
implemented.

Model outputs cannot be used as canonical truth, source-trust decisions,
rights clearance, malware safety, automatic identity merge, or automatic
master-index acceptance.
