# minimal_warc_wacz_capture_listing_v0

Synthetic WARC/WACZ capture listing. No archived content is fetched and no WARC payload is read.

This example is synthetic and public-safe. It includes no raw payload, no real extracted records, no source-cache or evidence-ledger records, no candidate promotion, no index mutation, no external calls, no OCR/transcription output, and no execution.
