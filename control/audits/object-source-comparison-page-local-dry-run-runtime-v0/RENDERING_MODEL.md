# Rendering Model

The dry-run renderer is local-only.

It:

- Reads approved examples only.
- Escapes HTML.
- Includes public-safe summaries only.
- Includes evidence/source references as labels only.
- Includes candidate, provisional, conflict, and gap warnings.
- Includes disabled action warnings.
- Uses no external CSS, JavaScript, or assets.
- Embeds no raw payloads.
- Claims no truth, rights clearance, malware safety, source trust, or installability.
- Writes no public site artifacts.

Rendered previews are audit-local artifacts or report fields, not hosted pages.
