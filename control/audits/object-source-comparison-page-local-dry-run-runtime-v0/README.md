# Object/Source/Comparison Page Local Dry-Run Runtime v0

P103 implements a local-only dry-run page runtime for approved repo examples.
It loads object, source, comparison, and P103 page preview records, classifies
them, and can render deterministic text, HTML, and JSON previews.

No public routes, hosted routes, API routes, public-search integration,
source-cache reads, evidence-ledger reads, live source calls, connector
execution, candidate promotion, downloads, uploads, installs, execution,
telemetry, accounts, or index mutation are enabled.
