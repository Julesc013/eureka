# Runtime Scope

Implemented:

- Local dry-run page record loader.
- Object/source/comparison page classifier.
- Public-safe HTML/text/JSON preview renderer.
- Local dry-run report builder.
- CLI dry-run script.
- Dry-run report validator.
- Tests.

- Public routes.
- Hosted routes.
- API routes.
- Public search integration.
- Page database.
- Persistent page store.
- Live source calls.
- Source cache reads or writes.
- Evidence ledger reads or writes.
- Candidate promotion.
- Index mutation.
- Downloads, uploads, installers, or execution.
- Telemetry or accounts.
