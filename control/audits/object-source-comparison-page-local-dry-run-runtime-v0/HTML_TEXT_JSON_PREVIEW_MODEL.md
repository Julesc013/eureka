# HTML/Text/JSON Preview Model

Text preview:

- Plain text and old-client safe.
- Includes title, page kind, status, lane, source/evidence summary posture,
  conflicts/gaps, disabled actions, and limitations.

HTML preview:

- Escaped HTML.
- No JavaScript required.
- No external assets.
- No form submission.
- No live links that fetch data.
- No download, install, upload, or execute controls.

JSON preview:

- Deterministic public-safe summary object.
- No private data.
- Hard booleans remain false except `local_dry_run`.
