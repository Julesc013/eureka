# Article/Scan Fixture Pack v0

This audit pack records the bounded Article/Scan Fixture Pack v0 milestone.

The pack targets `article_inside_magazine_scan`, the remaining archive-resolution
hard eval capability gap after More Source Coverage Expansion v1. It adds tiny
synthetic/recorded article-scan fixture text and page-range metadata so Eureka
can represent an article-like segment inside a scanned magazine-like parent
item.

This is fixture and eval support only. It does not add live source calls,
Internet Archive scraping, magazine scraping, OCR engines, PDF parsing, image
ingestion, real magazine scans, copyrighted article text, arbitrary local
filesystem ingestion, fuzzy/vector/LLM retrieval, Rust behavior, native apps, or
production deployment behavior. External baselines remain pending manual
observation.
