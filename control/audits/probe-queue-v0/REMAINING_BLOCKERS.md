# Remaining Blockers

- Runtime probe queue storage is intentionally not implemented.
- Probe execution is intentionally not implemented.
- Source cache and evidence ledger runtime remain future milestones.
- Candidate index and candidate promotion policy remain future milestones.
- Live-network probes require future approval, source policy, rate limits,
  timeout, backoff, circuit breaker controls, and operator setup.
- Hosted backend deployment remains operator-gated.
