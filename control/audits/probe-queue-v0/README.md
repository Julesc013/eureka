# Probe Queue v0

P63 defines Probe Queue v0 as a contract-only Query Intelligence Plane layer.
It creates governed schemas, synthetic public-safe examples, validators, docs,
and audit evidence for future probe planning items.

It does not implement a runtime queue, persistent queue, worker, connector
execution, source cache mutation, evidence ledger mutation, candidate-index
mutation, local-index mutation, master-index mutation, telemetry, external
source calls, or live probes.
