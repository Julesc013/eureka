# Runtime Boundary

P91 does not implement npm connector runtime. No live npm calls occur. No npm registry endpoints are called. No package metadata, version metadata, dist-tag metadata, or tarball metadata is fetched. No tarballs or package files are downloaded. No packages are installed. No npm/yarn/pnpm commands are invoked. No dependency resolution is performed. No npm audit is performed. No package archives are inspected. No lifecycle scripts are executed. No source-sync jobs execute. No source-cache records are written. No evidence-ledger records are written. No public search route calls npm. No arbitrary package or registry fetch exists. No downloads/mirroring/file retrieval exist. No credentials or npm tokens are configured. No telemetry is enabled. No indexes are mutated.
