# Runtime Scope

Implemented:

- local dry-run pack discovery
- local dry-run pack candidate classifier
- validator command adapter for approved repo validators
- dry-run import report builder
- dry-run diff/effect summary builder
- CLI dry-run script
- dry-run report validator
- tests

Not implemented:

- authoritative pack import
- real pack staging/quarantine
- promotion runtime
- public contribution intake
- upload/admin endpoints
- database tables
- persistent import queues
- source cache writes
- evidence ledger writes
- candidate index writes
- public/local/master index mutation
- execution of pack contents
- URL fetching from pack contents
- hosted runtime
- telemetry/accounts/downloads/uploads/installers
