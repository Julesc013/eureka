# Checked And Not Checked Model

Checked scope records explicit indexes, source families, capabilities, and snapshot refs. Not-checked scope records disabled, approval-gated, operator-gated, unavailable, or unimplemented areas.

P66 is contract-only. It adds no runtime known absence pages, no telemetry, no persistent query logging, no live probes, no source cache or evidence ledger runtime, no candidate promotion, and no public/master/local index mutation.
