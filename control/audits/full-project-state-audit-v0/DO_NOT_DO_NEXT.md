- Do not implement relay runtime or open sockets.
- Do not implement HTTP/FTP/SMB/AFP/NFS/WebDAV/Gopher/TLS/protocol translation.
- Do not implement live probes, call Internet Archive, call Google, scrape, crawl, fetch URLs, or automate external searches.
- Do not create Visual Studio or Xcode projects unless the human explicitly approves native skeleton implementation.
- Do not implement native GUI behavior, FFI, Rust runtime wiring, local cache runtime, private ingestion, telemetry, accounts, cloud sync, downloads, installers, package-manager integration, executable launch, or malware scanning.
- Do not claim rights clearance or executable safety.
- Do not deploy, change DNS, add CNAME, add provider config, or claim GitHub Pages deployment success without checked evidence.
- Do not weaken hard evals or fabricate external baseline observations.
- Do not treat policies and contracts as implemented runtimes.
