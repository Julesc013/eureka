# Example Candidate Record Review

P64 adds four synthetic, public-safe examples:

- `minimal_object_candidate_v0`: provisional software-version candidate.
- `minimal_evidence_candidate_v0`: provisional compatibility-claim candidate.
- `minimal_absence_candidate_v0`: scoped absence candidate with no broad
  non-existence claim.
- `minimal_conflict_candidate_v0`: possible duplicate identity candidate that
  preserves conflict for review.

All examples keep raw query retention disabled, live source calls disabled,
live probes disabled, payload actions disabled, accepted truth false, promotion
false, and all mutation flags false.
