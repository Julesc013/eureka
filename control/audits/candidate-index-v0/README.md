# Candidate Index v0

P64 defines Candidate Index v0 as a contract-only Query Intelligence Plane
milestone.

The pack proves the repo now has governed candidate record schemas, synthetic
examples, validators, dry-run stdout tooling, docs, tests, and inventory
metadata for provisional candidates. It does not add a runtime candidate index,
candidate promotion, public-search candidate injection, source-cache writes,
evidence-ledger writes, local-index mutation, master-index mutation, live
probes, telemetry, or external calls.

Required evidence is split across the files in this directory. The structured
report is `candidate_index_report.json`.
