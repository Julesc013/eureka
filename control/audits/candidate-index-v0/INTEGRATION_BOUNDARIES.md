# Integration Boundaries

P64 does not wire candidate records into public search, runtime storage, source
cache, evidence ledger, probe queue, search needs, result cache, miss ledger,
or master index.

Allowed P64 integration is limited to:

- governed contracts
- synthetic examples
- stdout-only dry-run helper
- validators
- tests
- docs
- AIDE and command metadata

Future integration requires P65 promotion policy plus privacy, poisoning,
source/evidence, and review gates.
