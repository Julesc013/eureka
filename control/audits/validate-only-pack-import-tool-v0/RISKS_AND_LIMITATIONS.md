# Risks And Limitations

- Detailed privacy, rights, and risk fields are conservative summaries of
  validator output.
- Report success means validators passed; it is not proof that claims are true.
- The tool does not stage packs for inspection.
- The tool does not build local indexes from pack contents.
- The tool does not submit contribution candidates.
- The tool does not call AI models or source APIs.
- Unknown roots are rejected safely, but the tool does not discover pack roots
  automatically.

The safe next step for user-local state is a separate Local
Quarantine/Staging Model v0 planning milestone or human-operated manual
observation work, depending the chosen path.
