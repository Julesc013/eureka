# Scope

Future skeleton scope, if explicitly approved later:

- Target OS: Windows 7 SP1+ x64
- UI stack: WinForms
- Runtime target: .NET Framework 4.8
- Toolchain: Visual Studio 2022
- Architecture target: x64
- Data mode: read-only static public data and seed snapshot metadata
- Network mode: no network by default
- Backend mode: no live backend required
- Source mode: no live probes, no external calls, no URL fetches
- Runtime mode: no downloads, installers, local cache, telemetry, accounts,
  relay, or Rust FFI

The future skeleton may display:

- app title
- local static data load status
- source summary
- eval summary
- demo query list
- limitations
- evidence/provenance placeholder panel

This planning milestone does not implement any of those screens.
