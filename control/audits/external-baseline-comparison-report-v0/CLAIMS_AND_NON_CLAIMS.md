# Claims and Non-Claims

This report does not claim Eureka beats Google, Archive.org, GitHub, PyPI, npm, Software Heritage, or any search engine.

Fixture-backed search quality does not prove real-world search superiority. Local_index_only search is not live-source search. External observations must be manual and explicit.

No observations were fabricated. No comparisons were fabricated. No external calls were performed. No production readiness claim is made.
