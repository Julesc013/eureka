# Delta Limitations

This audit is intentionally narrow.

Limitations:

- It measures local fixture/index behavior, not global web recall.
- It does not compare against Google, Internet Archive, or other systems because no manual observations are recorded.
- It does not claim mature hosted relevance.
- It does not claim broad source coverage.
- It does not claim rights clearance, malware safety, or executable trust.
- It does not treat subjective review/description notes as canonical compatibility truth.
- It does not add live calls, crawling, scraping, URL fetching, downloads, installs, uploads, accounts, telemetry, local path search, or hosted deployment.
- Exact failure-mode deltas are unavailable because the P32 pre-expansion failure-mode baseline is not committed as machine-readable JSON.
- Exact full per-query baseline rows are unavailable; query movement is limited to P32 selected targets and current expected-status evidence.
