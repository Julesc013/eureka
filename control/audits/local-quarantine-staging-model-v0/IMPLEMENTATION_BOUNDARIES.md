# Implementation Boundaries

This milestone does not implement:

- staging runtime
- `.eureka-local/` runtime state
- pack copying
- source/evidence/index/contribution pack import
- master-index queue import
- local index mutation
- runtime source registry mutation
- public-search mutation
- upload or submission
- moderation UI
- accounts or identity
- AI provider runtime or model calls
- live source connectors
- relay runtime
- native clients
- snapshot reader runtime

The only local-state filesystem change is preventive `.gitignore` policy.
