# Model Summary

Local quarantine/staging is the future private area after Validate-Only Pack
Import Tool v0 and before any staged inspection, local index candidate mode,
contribution export, or master-index review.

The model keeps staged metadata local/private by default. Staged records remain
claims, candidates, summaries, issues, or review notes. They are not canonical
truth and they do not affect public search or the master index.

No staging runtime exists in this milestone.
