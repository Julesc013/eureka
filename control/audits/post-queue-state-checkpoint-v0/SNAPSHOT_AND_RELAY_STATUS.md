# Snapshot And Relay Status

Signed Snapshot Format v0 is present as a contract and tiny repo seed example under `snapshots/examples/static_snapshot_v0/`. Required manifests, copied public data summaries, checksum file, and signature-placeholder documentation are validated.

There are no real signing keys, production signatures, production signed releases, executable downloads, or public `/snapshots/` route.

Relay is not implemented. No FTP, SMB, WebDAV, local LAN bridge, proxy, native client, or executable mirror exists. Relay Surface Design v0 is the recommended next Codex-safe design milestone.
