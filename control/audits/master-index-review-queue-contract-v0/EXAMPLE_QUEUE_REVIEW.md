# Example Queue Review

The example queue under `examples/master_index_review_queue/minimal_review_queue_v0/`
references the synthetic Contribution Pack v0 example. It has one queue entry
with `ready_for_review` validation and `human_review_required` review status.

The decision is `defer`, with public claims disabled and limitations recorded.
No real external observation, user identity, hosted queue state, or master-index
mutation is represented.
