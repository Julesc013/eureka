# PyPI Metadata Connector Runtime Planning v0

P90 plans a future PyPI metadata connector runtime. It keeps runtime blocked because P74 is present but `connector_approved_now` remains false and operator gates for package identity, dependency metadata caution, token/auth, source policy, User-Agent/contact, rate limit, timeout, cache, evidence attribution, package download/install/dependency boundaries, and kill switch are still pending.

This pack is planning-only. It makes no PyPI API calls, invokes no package manager, adds no live connector runtime, writes no source-cache or evidence-ledger records, enables no public-search live fanout, uses no tokens, downloads no package files, performs no dependency resolution, and mutates no candidate, public, local, runtime, or master index.
