# Privacy, Rights, And Risk Policy

Local staging is local/private by default. Public-safe staged records remain
candidates and still require review before any contribution or public export.

The manifest must not contain credentials, API keys, secrets, private keys,
unredacted private local paths, executable payloads, raw databases, raw caches,
raw private files, or long copyrighted payload dumps.

Manifest validity is not rights clearance, malware safety, source trust,
canonical truth, public-search acceptance, local index acceptance, or
master-index acceptance.
