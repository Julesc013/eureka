# Future Staged Inspector Path

Staged Pack Inspector v0 should come next. It should remain inspection-only and
read Local Staging Manifest v0 examples or future local-private manifests
without staging, importing, indexing, uploading, mutating runtime state,
exposing relay data, exporting snapshots, or touching the master index.

The inspector should report candidate posture, provenance, validation report
links, privacy/rights/risk summaries, issues, counts, and available future
delete/reset/export actions.
