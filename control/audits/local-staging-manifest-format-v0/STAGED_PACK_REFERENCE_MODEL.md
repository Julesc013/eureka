# Staged Pack Reference Model

Each staged pack reference records:

- staged pack reference ID
- pack ID and pack version
- pack type
- pack checksum
- pack root policy
- validation status
- privacy classification
- rights classification
- risk classification
- limitations

Supported pack types are source packs, evidence packs, index packs,
contribution packs, master-index review queues, and AI output bundles.

A staged pack reference is a local candidate reference. It is not import,
canonical acceptance, public search eligibility, local index mutation, upload,
or master-index acceptance.
