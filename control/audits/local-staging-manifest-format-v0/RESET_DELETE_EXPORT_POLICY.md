# Reset, Delete, And Export Policy

The manifest format requires future policy flags for:

- deleting one staged pack
- clearing all staged state
- exporting the manifest
- exporting a public-safe report view
- forbidding private data export by default
- requiring confirmation for irreversible deletes

These fields prepare future tooling. This milestone does not implement list,
inspect, delete, reset, export, staging, or quarantine runtime behavior.
