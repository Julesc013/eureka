# Source Gap Matrix

| Query family | Current status | Source families needed | Fixture families needed first | Likely connector later | Hard tests to add | Do not do |
| --- | --- | --- | --- | --- | --- | --- |
| Latest-compatible browser release queries | Mostly `source_gap` | vendor release archives, old release notes, package mirrors | recorded release metadata for Firefox, Chrome, Opera, VLC | vendor archive or curated release-index connector | last-compatible version with evidence and date constraints | Do not infer latest-compatible by fuzzy text alone |
| Windows 7 app queries | Mostly `source_gap` | software catalogs, portable app archives, release mirrors | recorded app catalog excerpts and archived download metadata | curated software catalog connector | object-type fit and compatibility evidence checks | Do not claim global recall or app-store coverage |
| Windows 98/2000/XP driver queries | Mostly `source_gap` | vendor support pages, support CDs, driver packs | recorded vendor page and support-media manifests | recorded support-media connector, later local-files connector | driver identity plus platform compatibility fixture | Do not fetch live driver sites or execute installers |
| Article-inside-scan queries | `source_gap` plus decomposition/OCR gaps | magazine scans, OCR text, issue metadata | recorded OCR snippets and issue manifests | Internet Archive recorded fixture connector later | article title/source/issue/page evidence checks | Do not scrape Internet Archive or claim OCR coverage |
| Manual/documentation queries | Mostly `source_gap` | manuals, PDFs, readmes, support docs | recorded PDF/manual metadata plus text snippets | recorded documentation source connector | manual exact-object and source-family evidence tests | Do not present outer container as useful if a PDF/manual is needed |
| Dead vendor link queries | Mostly `source_gap` | archived pages, Memento/Wayback snapshots | recorded memento snapshot metadata | Wayback/Memento recorded fixture connector | dead-link resolution with timestamped evidence | Do not add live crawling or browser automation |
| Package/container/member queries | `capability_gap` and `source_gap` | ISO/ZIP/package manifests, member text | local fixture packages with representative nested members | local-files fixture connector, later archive container connector | smallest-member actionable-unit checks | Do not default to parent ISO/ZIP when member is requested |
| Vague identity queries | Mostly `source_gap` | icon/name catalogs, software directories, screenshots | recorded identity clue fixtures | curated software identity connector | ambiguity and disagreement preservation checks | Do not add fuzzy/vector/LLM identity matching yet |
| Source-code/package-release queries | Mixed covered and gap | GitHub releases, source archives, Software Heritage | more recorded release and source snapshot fixtures | Software Heritage recorded fixture connector | source tarball and release-asset differentiation | Do not add live GitHub or SWH calls in required tests |
| Negative/absence queries | Mixed `capability_gap`/`unknown` | checked-source summaries | absence fixtures with known checked lanes | none first | absence reason completeness and no-private-path tests | Do not turn absence into a fake positive |
