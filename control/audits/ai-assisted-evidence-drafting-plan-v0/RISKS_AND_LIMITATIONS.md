# Risks And Limitations

- A valid typed AI output can still be wrong.
- A candidate can be mistaken for evidence truth if UI or docs are unclear.
- Remote providers can leak private data unless future policy blocks or gates
  context carefully.
- Synthetic examples do not prove real extraction quality.
- The plan does not implement review UI, provider selection, consent,
  credential storage, prompt logging, candidate export, or runtime drafting.
