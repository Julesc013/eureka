# Example Flow Review

The example flow under
`examples/ai_assisted_drafting/minimal_drafting_flow_v0/` is synthetic. It
contains:

- a public-safe input context
- a hand-authored typed AI output fixture
- a review-required evidence candidate
- a review-required contribution candidate
- checksums for all example files

No model call was performed. No provider runtime was loaded. No candidate was
accepted as evidence, contribution, public search data, local index data, or
master-index data.
