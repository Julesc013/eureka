# Runtime Boundaries

This milestone does not implement AI-assisted drafting runtime. It does not
call OpenAI, Anthropic, Ollama, LM Studio, local model servers, remote model
APIs, package registries, search engines, Internet Archive, Wayback, Software
Heritage, SourceForge, GitHub APIs, or external web APIs.

It does not fetch URLs, scrape, crawl, open sockets, add browser automation,
import AI output, mutate local indexes, mutate public search, mutate runtime
source registry state, upload, submit, or mutate the master index.
