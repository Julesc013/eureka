# Source Cache Output Plan

Future source-cache record kinds:

- github_repository_release_summary
- github_release_metadata_summary
- github_release_asset_metadata_summary
- github_release_tag_date_summary
- github_latest_release_summary

Required future constraints:

- metadata summaries only
- no release asset downloads
- no source archive downloads
- no raw payload dumps
- no raw blob/file fetches
- no repository clone
- no executable payloads
- no private repository access
- no private data
- source attribution
- repository identity review status
- freshness/invalidation policy
- source policy ref
- rate-limit/response metadata summary only
