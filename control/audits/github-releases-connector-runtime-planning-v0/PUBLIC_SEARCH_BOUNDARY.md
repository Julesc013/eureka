# Public Search Boundary

Public search must not call GitHub live.

Public search must not accept arbitrary owner/repo parameters for live fetch. Public query parameters must not choose GitHub live mode and must not provide arbitrary repository refs for live fetch.

Future public search may read public index/source cache summaries only after review. Future result cards may show source-cache/evidence refs only after future runtime and review.

Static site must not claim live GitHub Releases search until runtime and deployment evidence exist.
