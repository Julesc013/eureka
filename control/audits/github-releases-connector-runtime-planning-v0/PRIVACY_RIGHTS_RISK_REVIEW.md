# Privacy, Rights, And Risk Review

The future connector remains metadata-only and requires repository identity review.

P89 includes:

- no private repositories
- no token-required repositories
- no repository clone
- no release asset download
- no source archive download
- no raw file/blob fetch
- no downloads
- no file retrieval
- no mirroring
- no install or execute
- no rights clearance
- no malware safety
- no asset safety claim
- no installability claim
- no private account access
- no credentials
- no raw payload dumps
- no private paths or secrets

Source terms review remains pending unless separately approved later.
