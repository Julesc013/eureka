# Do Not Implement Yet

P101 does not implement:

- connector runtimes
- live external source calls
- public-search fanout
- arbitrary URL, repository, package, or SWHID fetch
- downloads
- source archive fetch
- repository clone
- package install
- dependency resolution
- lifecycle script execution
- source-code content fetch
- source-cache or evidence-ledger authoritative writes
- candidate, public, local, runtime, or master index mutation
- credentials or tokens
- telemetry or accounts
- production-readiness claims

Plain prohibitions: no connector runtimes, no live external source calls, no public-search fanout, and no credentials or tokens.
