# Output Types Review

Valid output types remain bounded candidate/draft forms:

- interpreted query candidates
- alias candidates
- metadata claim candidates
- compatibility claim candidates
- review claim candidates
- member path candidates
- source match candidates
- identity match candidates
- explanation drafts
- OCR cleanup candidates
- absence explanation drafts
- contribution draft candidates

The validator rejects unsupported task or output types and blocks authority
outputs such as rights-clearance decisions, malware-safety decisions, source
trust decisions, automatic identity merges, and automatic master-index
acceptance.
