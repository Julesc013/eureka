# Hosted Public Search Wrapper v0

P54 adds a deployable stdlib wrapper for Eureka public search in
`local_index_only` mode. The wrapper is local-rehearsed only; no hosted backend
deployment, provider API call, DNS, credentials, live probes, downloads,
uploads, accounts, telemetry, arbitrary URL fetching, source connector runtime,
AI runtime, index mutation, pack import, or staging runtime is added.
