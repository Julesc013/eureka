# Index Pack Contract v0 Audit

This audit pack records the implementation of Index Pack Contract v0.

The milestone is contract, validation, and example-pack work only. It does not
implement index import, merge, upload, public hosted ingestion, raw cache
export, raw SQLite export, live connectors, executable plugins, downloads,
installers, or master-index acceptance.
