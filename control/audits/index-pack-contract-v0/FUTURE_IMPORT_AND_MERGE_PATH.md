# Future Import And Merge Path

Future import or compare tooling must be planned separately.

Required future gates include:

- source/evidence provenance comparison
- checksum verification
- redaction review
- source-family trust policy
- duplicate and conflict handling
- local-private data rejection
- rights/access review
- result-card projection rules
- master-index review queue policy
- operator-visible rollback or quarantine behavior

P36 implements none of those runtime or hosted workflows.
