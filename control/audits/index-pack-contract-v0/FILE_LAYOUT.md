# File Layout

Required Index Pack v0 files:

- `INDEX_PACK.json`
- `README.md`
- `PRIVACY_AND_RIGHTS.md`
- `CHECKSUMS.SHA256`
- `index_summary.json`
- `source_coverage.json`
- `record_summaries.jsonl`

Optional files:

- `field_coverage.json`
- `query_examples.jsonl`
- `evidence_references.jsonl`
- `source_references.jsonl`
- `compatibility_summary.json`
- `member_summary.json`
- `absent_or_gap_summary.json`
- `manifests/`
- `tests/`
- `licenses/`
- `references/`

The example pack uses the required files plus `field_coverage.json` and
`query_examples.jsonl`.
