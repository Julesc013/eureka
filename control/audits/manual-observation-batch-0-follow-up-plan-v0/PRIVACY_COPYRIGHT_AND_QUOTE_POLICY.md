# Privacy, Copyright, and Quote Policy

Human operators must not record:

- Private account data
- Private search history
- Credentials, tokens, passwords, or API keys
- IP addresses, session IDs, browser identifiers, or account IDs
- Private URLs, private repository names, private package names, or user identifiers
- Long copyrighted excerpts
- Raw copyrighted payload dumps
- Executable payloads or real binaries

Prefer paraphrase plus a source reference. Quotes should be short, necessary, and bounded. Mark uncertainty and conflicts explicitly.

Observation records must not claim:

- Rights clearance
- Malware safety
- Dependency safety
- Installability
- Production search superiority
- Global absence

External observations are evidence inputs, not truth.
