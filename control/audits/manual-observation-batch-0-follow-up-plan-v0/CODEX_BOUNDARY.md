# Codex Boundary

Codex may:

- Inspect repo-local files.
- Create templates, worksheets, docs, validators, and reports.
- Run local validators and comparison/status scripts that perform no external calls.
- Summarize missing, pending, or invalid records.
- Provide repair guidance.

Codex must not:

- Perform manual observations.
- Browse external sites.
- Call Google, Internet Archive, Wayback, GitHub, PyPI, npm, Software Heritage, or external search/source APIs.
- Scrape, crawl, or fetch external pages.
- Fabricate external results, screenshots, dates, quotes, or comparison findings.
- Mark pending observations complete.
- Claim comparison eligibility without valid observations.
- Mutate public/local/runtime/master indexes, source cache, evidence ledger, or candidate index.

Exact boundary summary: Codex must not perform manual observations, must not browse external sites, and must not fabricate external results.

This pack is a plan and workflow aid only.
