Human-facing result quality guidance:

- `exact`: the result directly identifies the requested artifact, item, version, page, driver, or actionable software target.
- `partial`: the result is relevant but incomplete, parent-level only, ambiguous, missing compatibility, or missing the smallest actionable unit.
- `near_miss`: the result is adjacent but likely not the requested target.
- `none`: no useful result was visible in the manual observation scope.
- `not_observed`: the operator did not complete the observation.
