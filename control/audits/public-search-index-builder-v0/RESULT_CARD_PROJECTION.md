# Result Card Projection

The runtime reconstructs `IndexRecord` values from public index documents and
uses the existing public result-card projection. Result cards continue to show:

- source and evidence summaries
- compatibility posture
- user-cost summary
- allowed read/inspect/provenance actions
- blocked download/upload/install/execute actions
- rights and malware-safety caveats
- local-index-only limitations

No result card claims rights clearance, malware safety, hosted deployment, or
global recall.
