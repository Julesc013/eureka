# Safety And Privacy Review

P55 hard booleans:

- live sources used: false
- external calls performed: false
- private paths detected: false
- executable payloads included: false
- downloads enabled: false
- uploads enabled: false
- local paths enabled: false
- arbitrary URL fetch enabled: false
- master index mutated: false
- pack import performed: false
- AI runtime enabled: false

Validators check for absolute/private path markers, secret-like tokens, live
source flags, and unsafe enabled actions.
