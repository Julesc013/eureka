# Deep Extraction Contract v0

P95 defines Deep Extraction Contract v0 for future metadata-first inspection of containers, archives, packages, disc images, WARC/WACZ files, source bundles, installers, scanned volumes, and nested records.

This audit pack is contract/schema/example/validator work only. It implements no extraction runtime and performs no unpacking, OCR, transcription, execution, download, source fetch, source-cache write, evidence-ledger write, candidate promotion, public index mutation, local index mutation, runtime index mutation, or master-index mutation.
