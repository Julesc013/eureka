# Manifest And Metadata Extraction Model

Manifest summaries may describe package manifests, installer manifests, disc manifests, WARC indexes, WACZ manifests, METS, ALTO, IIIF manifests, checksums manifests, SBOM, SPDX, PURL metadata, or unknown manifests.

Manifest extraction is summary-only. Raw manifests are excluded unless a future reviewed policy allows safe excerpts. Manifest metadata is not rights clearance, malware safety, dependency safety, installability, or source completeness.
