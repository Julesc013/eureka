# No Runtime And No Mutation Policy

P95 does not implement extraction runtime.

No files are opened. No archives are unpacked. No payloads, installers, scripts, macros, package managers, emulators, or VMs are executed. No OCR or transcription is performed. No URLs are fetched. No live sources are called. No external APIs are called. No database tables, persistent queues, extraction stores, runtime routes, or pack-import integration are added.

No source cache, evidence ledger, candidate index, public index, local index, runtime index, or master index is mutated. No candidates are promoted.
