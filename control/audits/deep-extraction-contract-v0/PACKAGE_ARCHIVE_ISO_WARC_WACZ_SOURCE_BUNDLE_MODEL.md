# Package Archive ISO WARC WACZ Source Bundle Model

The contract covers package archives, wheels, sdists, npm tarballs, ISO/disk images, WARC/WACZ capture containers, source bundles, repository snapshots, installers, PDFs, and scanned volumes as metadata-only future targets.

Forbidden now:

- package download or install
- dependency resolution
- source-code download
- repository clone
- WARC/WACZ payload fetch
- ISO/disk image reads
- installer execution
- archive unpacking
- executable inspection beyond metadata-risk labels

Source bundles and package metadata are not source-code safety, rights clearance, dependency safety, or installability proof.
