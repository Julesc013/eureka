# Nested Recursion And Depth Limit Model

Nested extraction is disabled now.

Future recursion requires:

- explicit maximum depth
- maximum member count
- maximum total uncompressed size
- maximum member size
- maximum runtime
- maximum text bytes
- OCR page limits
- decompression bomb guard
- archive traversal guard
- symlink guard
- temporary workspace and cleanup
- operator approval

Tier 4 requires sandbox approval. Tier 5 requires explicit on-demand approval for each deepening step.
