# Staging Report Path Contract v0

This audit pack records Staging Report Path Contract v0.

The slice is planning/contract/governance only. It defines report path rules
for future validate-only reports, staging reports, and local staging manifests.
It does not implement staging runtime, create local report directories, copy
pack files, import pack contents, mutate local indexes, mutate public search,
upload, submit, or mutate the master index.
