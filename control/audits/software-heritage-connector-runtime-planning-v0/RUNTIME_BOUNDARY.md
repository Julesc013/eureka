# Runtime Boundary

- P92 does not implement Software Heritage connector runtime.
- No live Software Heritage calls occur.
- No Software Heritage API endpoints are called.
- No SWHIDs are resolved live.
- No origins, visits, snapshots, releases, revisions, directories, or content blobs are fetched.
- No source files or source archives are downloaded.
- No repositories are cloned.
- No Git remotes are fetched.
- No source code is inspected or executed.
- No source-sync jobs execute.
- No source-cache records are written.
- No evidence-ledger records are written.
- No public search route calls Software Heritage.
- No arbitrary SWHID/origin/repository fetch exists.
- No downloads/mirroring/file retrieval exist.
- No credentials or Software Heritage tokens are configured.
- No telemetry is enabled.
- No indexes are mutated.
