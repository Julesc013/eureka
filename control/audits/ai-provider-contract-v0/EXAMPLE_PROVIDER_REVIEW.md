# Example Provider Review

The example provider is `example.disabled_stub_provider_v0`.

It is synthetic, non-operational, and disabled by default. It requires no
network, credentials, local filesystem access, prompt logging, output logging,
telemetry, or cache runtime.

The example includes two hand-authored typed outputs:

- an alias candidate
- an explanation draft

Both require review and prohibit canonical truth, rights clearance, malware
safety, and automatic acceptance uses.
