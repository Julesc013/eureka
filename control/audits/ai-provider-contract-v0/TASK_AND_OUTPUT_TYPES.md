# Task And Output Types

Allowed task types are suggestion tasks such as query interpretation, alias
suggestions, metadata extraction suggestions, compatibility candidates, member
path extraction, source matching, duplicate identity candidates, explanation
drafts, OCR cleanup suggestions, absence explanation drafts, and contribution
pack draft assistance.

Allowed outputs are typed candidates or drafts. They must include limitations,
provider provenance, review requirements, and prohibited uses.

Forbidden roles include canonical truth decisions, source-trust final
decisions, rights clearance, malware safety, automatic identity merge,
automatic master-index acceptance, silent private data processing, arbitrary
URL fetch, credential extraction, installer/download execution, and telemetry
collection.
