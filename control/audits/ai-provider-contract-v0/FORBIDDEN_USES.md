# Forbidden Uses

AI Provider Contract v0 forbids using AI output as:

- canonical truth
- source-trust final decision
- rights clearance
- malware safety
- automatic identity merge
- automatic master-index acceptance
- silent private data processing
- unbounded web browsing or arbitrary URL fetching
- credential extraction
- installer or download execution
- telemetry collection

The contract also forbids claiming production AI support or public-search AI.
