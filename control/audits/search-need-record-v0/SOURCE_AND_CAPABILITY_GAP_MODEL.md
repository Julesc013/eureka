# Source And Capability Gap Model

Search need records carry gap summaries such as source coverage gap,
capability gap, compatibility evidence gap, member access gap, representation
gap, query interpretation gap, disabled live probe, pending external baseline,
deep extraction missing, OCR missing, source cache missing, or unknown.

Suggested resolutions are future-only. P62 does not add sources, evidence
packs, probes, connectors, extraction, OCR, or parser runtime behavior.
