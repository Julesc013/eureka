# Source Family Selection

Selected fixture families:

- `wayback_memento_recorded`: dead vendor pages, archived release notes, archived download page metadata, and capture-locator evidence.
- `software_heritage_recorded`: source snapshot and SWHID-like provenance metadata.
- `sourceforge_recorded`: legacy FOSS project/release metadata and visual-description hints.
- `package_registry_recorded`: versioned package/release metadata and member readme hints.
- `manual_document_recorded`: manual, hardware-maintenance, and document metadata.
- `review_description_recorded`: review/description compatibility notes treated as subjective evidence.

The selection intentionally avoids live APIs and broad source sprawl. The next source work should split these fixture shapes into governed source/evidence/index pack contracts before any live-source approval.
