# Acceptance Criteria

Before implementation:

- search result explanation contract valid
- explanation examples valid
- public search contract valid
- public result-card contract valid
- public index/result envelope valid
- public search safety valid
- public search runtime integration audit valid
- privacy/redaction policy accepted
- user-facing copy policy accepted
- no-model/no-AI boundary accepted
- no-hidden-score/no-suppression policy accepted
- source/evidence/candidate boundary accepted
- ranking/merge/identity dependency handling accepted
- fallback/rollback model accepted
- hosted deployment evidence required for hosted mode
- operator approval recorded
