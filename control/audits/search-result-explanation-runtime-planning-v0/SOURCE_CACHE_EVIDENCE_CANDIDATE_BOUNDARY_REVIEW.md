# Source Cache, Evidence, And Candidate Boundary Review

The future explanation runtime may later read reviewed public-safe source and
evidence references from a public result envelope or public index document.

P106 does not read:

- authoritative source cache
- authoritative evidence ledger
- private source-cache records
- private evidence-ledger records
- candidate index records

P106 does not write:

- source cache
- evidence ledger
- candidate index
- public index
- local index
- master index

Source and evidence references are explanatory context, not accepted truth.
Candidate/provisional status must remain visible. Candidate promotion is not an
explanation concern and remains separately review-gated.
