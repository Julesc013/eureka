# API, Static, Lite, And Text Output Plan

API response future:

- structured explanation fields
- ordered component list
- warnings and limitations
- no hidden scores
- no raw private query
- no source-cache/evidence-ledger private records

Static/lite/text future:

- old-client-safe summary
- no JavaScript required
- no external assets
- visible caveats
- bounded length

HTML future:

- escaped output
- no external fetch
- no download, install, upload, or execute controls
- no live source links that trigger fetching

P106 adds none of these outputs to public search.
