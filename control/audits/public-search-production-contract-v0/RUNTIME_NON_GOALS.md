# Runtime Non-Goals

P53 does not implement:

- hosted backend deployment
- deployment provider configuration
- source connector runtime
- live probes
- external source API calls
- scraping or crawling
- model calls or AI runtime
- credentials or telemetry runtime
- accounts
- uploads
- downloads
- installers
- arbitrary URL fetching
- local/runtime/master index mutation
- pack staging or import
- production readiness claims
