# Public Search Production Contract v0

P53 freezes Eureka's first production-facing public search contract for the
future P54 hosted wrapper.

Scope: routes, request/response/error schemas, result-card alignment,
source-status/evidence/absence/status schemas, safety limits, static handoff,
and hosted-wrapper requirements.

Non-goals: backend deployment, source connector runtime, live probes, external
API calls, scraping, AI runtime, telemetry, accounts, uploads, downloads,
installers, index mutation, pack import, and production readiness claims.
