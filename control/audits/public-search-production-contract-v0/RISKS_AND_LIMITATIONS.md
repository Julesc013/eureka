# Risks And Limitations

Top risks:

- confusing local prototype behavior with hosted search
- allowing dangerous request parameters in P54
- emitting private paths or raw source payloads
- weak public error vocabulary
- result cards implying rights clearance or malware safety
- static pages hardcoding fake backend URLs
- missing hosted rate limiting
- missing operator kill switch
- source coverage gaps
- query intelligence still absent

P53 reduces schema ambiguity but does not remove hosted deployment, ops, source
coverage, or query-intelligence gaps.
