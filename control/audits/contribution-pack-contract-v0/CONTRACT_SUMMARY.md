# Contract Summary

Contribution Pack Contract v0 defines the first safe submission wrapper for
future review workflows. A contribution pack can reference source, evidence, and
index packs; carry public-safe candidate items; and record privacy, rights,
review, and checksum posture.

Contribution items are review candidates, not truth. The contract creates no
runtime path that can mutate source registries, local indexes, public search,
snapshots, hosted indexes, or a master index.
