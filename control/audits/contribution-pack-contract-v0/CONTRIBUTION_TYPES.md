# Contribution Types

Allowed contribution types include source-record candidates, evidence-record
candidates, index-coverage candidates, manual observations, metadata
corrections, alias suggestions, compatibility suggestions, member-path
suggestions, checksum observations, absence reports, dead-link reports, result
quality feedback, source-coverage reports, duplicate or identity candidates,
documentation hints, and review notes.

Allowed proposed actions are review requests only. They do not directly add,
update, deprecate, split, merge, link, or accept records.
