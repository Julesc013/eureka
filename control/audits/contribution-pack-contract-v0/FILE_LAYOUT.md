# File Layout

Required files:

- `CONTRIBUTION_PACK.json`
- `README.md`
- `PRIVACY_AND_RIGHTS.md`
- `CHECKSUMS.SHA256`
- `contribution_items.jsonl`

Optional files include referenced pack records, manual observations, metadata
corrections, alias suggestions, compatibility suggestions, member-path
suggestions, absence reports, result feedback, review notes, manifests, tests,
licenses, and references.

The example pack uses only tiny self-authored JSON/JSONL/Markdown files.
