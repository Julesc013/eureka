# Example Pack Review

Example pack:

`examples/evidence_packs/minimal_evidence_pack_v0/`

The example is synthetic and self-authored. It includes:

- four evidence records
- one source reference
- short snippets below 500 characters
- no executable payloads
- no private paths
- no live URL fetch requirement
- no rights-clearance or malware-safety claim

The example records describe metadata, compatibility, member-path, and absence
claims for an illustrative browser fixture.
