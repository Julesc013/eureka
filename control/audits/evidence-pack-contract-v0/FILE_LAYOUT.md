# File Layout

Required files:

- `EVIDENCE_PACK.json`
- `README.md`
- `RIGHTS_AND_ACCESS.md`
- `CHECKSUMS.SHA256`
- `evidence_records.jsonl`

Optional files include `source_references.jsonl`, claim-specific JSONL files,
`snippets/`, `references/`, `licenses/`, `tests/`, and `manifests/`.

The example pack uses only the required files plus `source_references.jsonl`.
