# Claim And Evidence Types

Allowed evidence kinds include source observations, metadata claims,
compatibility claims, member-path claims, checksum observations, version
observations, release-note observations, manual-document observations,
review-description observations, absence observations, identity candidates,
provenance notes, and actionability notes.

Allowed claim types include describes, mentions, supports/does-not-support
platform, requires runtime, has version, latest-known-for-platform, contains
member, has checksum, documents hardware/install steps, reports works/failure,
source missing, dead link observed, archived trace exists, same-as/variant
candidates, evidence for absence, and actionability hints.
