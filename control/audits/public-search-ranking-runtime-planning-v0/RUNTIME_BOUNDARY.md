# Runtime Boundary

P97 does not implement ranking runtime. No public search order is changed. No public search response is changed. No result suppression occurs. No hidden scores are added. No telemetry, popularity, user-profile, ad, or model signals are used. No DB tables are added. No persistent ranking store is added. No source-cache/evidence-ledger/candidate/public/master mutation occurs. No live source calls occur. No downloads/installers/execution are enabled.
