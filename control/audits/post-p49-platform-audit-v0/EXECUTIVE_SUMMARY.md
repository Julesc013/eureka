# Executive Summary

Eureka is a contract-governed, Python-reference, pre-product platform with a
validated static publication artifact, local/prototype public search, fixture
and recorded-fixture source coverage, pack contracts and validators, planning
for local staging, and optional AI-assistance contracts. It is not a hosted
public search service and does not have live connectors, pack import runtime,
AI runtime, accounts, uploads, downloads, telemetry, or production operations.

What is real:

- `implemented_static_artifact`: `site/dist`, generated public data, lite/text/files surfaces, static resolver demos, and GitHub Pages workflow configuration for `site/dist`.
- `implemented_local_prototype`: local public search routes and rehearsal, aggregate pack validation, validate-only pack import, staged pack inspection, public-alpha safe wrapper checks, and local CLI/API/demo surfaces.
- `implemented_runtime`: Python validators, eval runners, source-registry loading, typed AI output validation helper, and current local runtime seams.
- `contract_only`: public search contracts, result-card contract, live backend handoff, live probe gateway, pack contracts, review queue, native/snapshot/relay contracts, and AI provider contracts.
- `planning_only`: local quarantine/staging runtime, assisted drafting runtime, relay implementation, WinForms skeleton, and Rust local-index parity.

What is not real:

- hosted public search
- hosted backend deployment
- live Internet Archive, Wayback/CDX/Memento, GitHub Releases, PyPI, npm, Software Heritage, Wikidata/Open Library, or other source connectors
- external baseline observations
- pack import into runtime or master index
- public query learning runtime
- AI provider/model runtime
- native GUI or relay runtime
- Rust production runtime replacement
- production ops stack

Top blockers are GitHub Pages deployment evidence failure, zero external baseline observations, no hosted backend/ops stack, no live connector approval, no query intelligence contracts/runtime, and missing root community/security documents.
