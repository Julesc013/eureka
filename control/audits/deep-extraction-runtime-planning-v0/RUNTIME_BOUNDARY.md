# Runtime Boundary

P105 does not implement extraction runtime.

It adds:

- no extraction queue
- no extraction store
- no sandbox runtime
- no archive unpacker
- no OCR runtime
- no transcription runtime
- no file opening
- no archive unpacking
- no arbitrary path reads
- no URL fetching
- no payload execution
- no source/evidence/candidate/index mutation
- no mutation
- no public search integration
- no page integration
- no connector integration
- no pack import integration
