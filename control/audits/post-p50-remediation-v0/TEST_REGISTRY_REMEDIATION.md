# Test Registry Remediation

P51 registers the remediation validator and focused tests in:

```text
control/inventory/tests/test_registry.json
```

The validator/tests cover:

- required audit files
- JSON parseability
- required remediation item IDs
- root governance status
- GitHub Pages evidence honesty
- Cargo status
- pack validator CLI remediation
- next branch recommendation
- no product behavior expansion claims

The registry remains operating metadata only.
