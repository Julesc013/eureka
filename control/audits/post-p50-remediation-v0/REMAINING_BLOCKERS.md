# Remaining Blockers

- GitHub Pages deployment evidence requires operator settings and successful
  workflow evidence.
- License selection requires owner/legal decision.
- Cargo checks require Rust toolchain installation.
- External baseline observations require human operation.
- Hosted public search requires backend hosting, DNS/TLS, rate limits, privacy
  policy, logging policy, rollback, monitoring, and operator setup.
- Live connectors require explicit approval, source-specific policy, cache and
  evidence ledger design, and no-unbounded-probe enforcement.
- Query intelligence runtime remains unimplemented and should be planned before
  public query learning.
