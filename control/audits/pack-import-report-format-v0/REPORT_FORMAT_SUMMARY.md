# Report Format Summary

Pack import reports are local, durable records of validation activity. They
record inputs, validators, pack results, issue summaries, privacy/rights/risk
status, provenance, next actions, and hard false mutation fields.

Reports are not imports, staging records, local-index writes, uploads,
submissions, or master-index decisions. A successful report means the recorded
validation activity passed; it does not mean source truth, rights clearance,
malware safety, or public acceptance.
