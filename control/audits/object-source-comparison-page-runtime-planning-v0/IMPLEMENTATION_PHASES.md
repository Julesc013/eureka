# Implementation Phases

Phase 0: keep disabled.

- Complete planning and validation.

Phase 1: local dry-run page assembler over synthetic examples only.

- No runtime routes.

Phase 2: local read-only runtime routes from public index only.

- No hosted deployment.
- No source/evidence/candidate mutation.

Phase 3: hosted staging runtime with safe route IDs and blocked-parameter tests.

- No live source calls.

Phase 4: public alpha page runtime with object/source/comparison links from public search.

- No downloads/install/execution.

Phase 5: generated static page snapshots for high-value records and offline/lite clients.
