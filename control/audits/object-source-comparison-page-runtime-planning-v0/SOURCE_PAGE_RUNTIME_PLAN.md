# Source Page Runtime Plan

Input:

- source_id from source inventory, public index, or reviewed page record only

Output:

- HTML page
- text/lite page
- JSON API response

Sections:

- source identity
- source status
- coverage
- connector posture
- source policy/approval gates
- source cache/evidence ledger posture
- public search posture
- limitations/gaps
- rights/risk/action posture

Constraints:

- no connector calls
- no live source probing
- no source trust overclaim
- no download/mirror/install/execute
- no mutation
