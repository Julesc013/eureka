# Search Result Explanation Integration Status

Classification: `contract_only`.

Status:

- Explanation contract: present from P96.
- Runtime implementation: absent.
- Public search response integration: not integrated as a P96 runtime.
- Result explanation route status: absent.
- AI/model calls: disabled.
- Hidden scores: disabled.
- Mutation status: no source cache, evidence ledger, candidate index, public index,
  local index, or master index mutation.

Clarification:

Current result cards contain simple bounded fields such as `why_matched` and
`why_ranked`. They are not the P96 search-result-explanation runtime and do not
add hidden scores, AI answers, or response mutation from an explanation service.
