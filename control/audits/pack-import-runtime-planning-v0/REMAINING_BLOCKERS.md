# Remaining Blockers

- Operator approval is required before local dry-run runtime work.
- Pack import runtime is not implemented.
- Runtime quarantine/staging store is not implemented.
- Promotion runtime remains blocked and separate.
- Source-cache runtime approval is required before any source cache mutation.
- Evidence-ledger runtime approval is required before any evidence ledger mutation.
- Candidate/public/local/master index mutation remains blocked.
- Public contribution intake, upload endpoints, admin endpoints, accounts, telemetry, and moderation remain out of scope.
