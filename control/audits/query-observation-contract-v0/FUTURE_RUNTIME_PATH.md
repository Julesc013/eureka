# Future Runtime Path

Future runtime integration should happen only after privacy and poisoning
controls are governed. A later branch may add:

1. Shared Query/Result Cache v0
2. Search Miss Ledger v0
3. Search Need Record v0
4. Probe Queue v0
5. Candidate Index v0
6. Candidate Promotion Policy v0
7. Query Privacy and Poisoning Guard v0

Public queries may inform learning, but accepted truth must remain evidence and
validation gated.
