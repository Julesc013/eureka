# Implementation Boundaries

This milestone is planning and validation only.

It does not implement:

- pack import runtime
- source pack import
- evidence pack import
- index pack import
- contribution pack import
- master-index queue import
- arbitrary local directory scanning
- local staging directories
- local search mutation
- local index mutation
- canonical source registry mutation
- hosted/master index mutation
- upload or submission
- moderation UI
- accounts, auth, identity, telemetry, TLS, rate limiting, or process management
- executable plugin loading
- live source connectors or live probes
- URL fetching, scraping, crawling, or external web API calls
- downloads, installers, restores, or execution

Implementation work starts only after the planning validator and docs remain
green and a later milestone chooses a specific validate-only tool boundary.
