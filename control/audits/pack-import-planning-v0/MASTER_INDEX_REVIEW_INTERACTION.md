# Master Index Review Interaction

Pack import is local. Master-index review is separate. There is no automatic acceptance.

Future import may prepare contribution candidates, but it does not submit,
upload, accept, reject, supersede, or publish anything.

The future path is:

1. Validate a source/evidence/index/contribution pack.
2. Stage or inspect locally.
3. Optionally create a contribution pack or queue candidate in a later tool.
4. Review the candidate through Master Index Review Queue policy.
5. Accept, reject, defer, quarantine, or supersede through an auditable review
   decision.

accepted_public requires a review decision and remains limited,
provenance-bound, and non-authoritative. Import never creates automatic
acceptance and never writes the hosted/master index.
