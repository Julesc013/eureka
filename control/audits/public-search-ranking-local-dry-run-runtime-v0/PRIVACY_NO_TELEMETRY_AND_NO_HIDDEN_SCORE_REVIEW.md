# Privacy, No-Telemetry, And No-Hidden-Score Review

P107 rejects or forbids:

- raw private query input
- IP, account, session, or user-profile data
- telemetry signals
- popularity signals
- ad signals
- model signals
- hidden scores
- result suppression
- private paths
- private URLs
- local machine fingerprints

The committed examples are synthetic and public-safe.
