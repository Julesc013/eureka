# Comparison Page Schema

`contracts/pages/comparison_page.v0.json` requires identity, type, subjects, criteria, comparison matrix, identity/version/representation/source/evidence/compatibility/action/conflict/gap sections, projections, privacy, limitations, no-winner guarantees, no-runtime guarantees, and no-mutation guarantees. Hard booleans require runtime, live calls, winner claims, risky actions, and mutations to be false.
