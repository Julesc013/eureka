# Lite, Text, And Files Review

The P56 search handoff is projected into old-client surfaces:

- `site/dist/lite/search.html`
- `site/dist/text/search.txt`
- `site/dist/files/search.README.txt`

These surfaces are generated by `scripts/generate_compatibility_surfaces.py`.
They remain no-JS/static, state the backend-unconfigured status, include local
runtime instructions, and reference:

- `data/search_config.json`
- `data/public_index_summary.json`
- `data/search_handoff.json`

They do not expose downloads, uploads, live probes, accounts, local paths, or
hosted-search availability.
