# Search Page Review

`site/dist/search.html` is generated by `site/build.py` from governed site page
data and publication inventory inputs.

The generated page:

- uses semantic HTML and no required JavaScript;
- shows that hosted public search is not configured yet;
- explains that local/prototype `local_index_only` search and the generated
  public index exist but are not hosted by the static site;
- provides sample queries;
- links to status, sources, demo, lite, text, files, search configuration, and
  public index summary surfaces;
- includes local wrapper commands for local rehearsal;
- does not include a fake hosted backend URL.

The page includes a disabled search form for visual affordance only while
backend status is unconfigured. It does not submit to localhost or a fabricated
public URL.
