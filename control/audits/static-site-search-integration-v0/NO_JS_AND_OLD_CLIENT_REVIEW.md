# No-JS And Old-Client Review

The search handoff is compatible with static and older clients:

- `search.html` requires no JavaScript.
- `lite/search.html` uses simple HTML.
- `text/search.txt` provides a plain text projection.
- `files/search.README.txt` documents the file-tree surface.
- Public JSON data can be read directly by static, native, relay, or snapshot
  consumers later.

Base-path safety is preserved by relative links generated by the site builder.
The static pages do not include local absolute paths, scripts, credentials,
downloads, uploads, or live-probe claims.
