# Integration Boundaries

P61 is a contract/schema/docs/validation milestone.

Allowed in P61:

- governed schemas
- query-intelligence policy inventory
- synthetic examples
- validators and focused tests
- stdout-only dry-run helper
- docs and audit metadata

Not implemented in P61:

- persistent miss ledger
- runtime ledger writes
- telemetry
- public query logging
- search need creation
- probe enqueueing
- candidate index mutation
- result cache mutation
- local or master index mutation
- live probes or external source calls
