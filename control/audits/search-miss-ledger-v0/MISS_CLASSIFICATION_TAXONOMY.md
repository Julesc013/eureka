# Miss Classification Taxonomy

Allowed miss types:

- `no_hits`
- `weak_hits`
- `near_miss_only`
- `blocked_by_policy`
- `source_coverage_gap`
- `capability_gap`
- `compatibility_evidence_gap`
- `member_access_gap`
- `representation_gap`
- `query_interpretation_gap`
- `live_probe_disabled`
- `external_baseline_pending`
- `unknown`

Severity values are `info`, `low`, `medium`, and `high`.

Confidence values are `low`, `medium`, and `high`.

Every v0 entry must keep `global_absence_claimed` false.
