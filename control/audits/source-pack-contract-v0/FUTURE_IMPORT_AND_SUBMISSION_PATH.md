# Future Import And Submission Path

Future milestones must keep source-pack validation separate from source-pack
acceptance.

Expected future path:

1. Source pack validates locally.
2. A future import plan defines how a validated local pack may feed a controlled
   local index.
3. Evidence Pack Contract v0 defines reusable evidence semantics.
4. Index Pack Contract v0 defines portable index artifacts.
5. Contribution Pack Contract v0 defines reviewable submissions.
6. Master Index Review Queue Contract v0 defines public acceptance gates.

Before hosted or master-index acceptance, a pack must be reviewed for public
safety, rights/access posture, source identity, fixture scope, evidence quality,
privacy leakage, checksums, and disabled live behavior.

No acceptance queue or hosted submission exists in P34.
