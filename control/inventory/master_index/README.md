# Master Index Review Inventory

This inventory records contract-only governance for the future Eureka master
index review queue. It is not a hosted queue, upload service, account system,
moderation runtime, source/evidence/index/contribution pack import path, or
master-index writer.

- `review_queue_policy.json` defines accepted input pack types, required checks,
  prohibited inputs, and publication posture.
- `review_state_taxonomy.json` defines validation and review states.
- `acceptance_requirements.json` defines future acceptance gates.
