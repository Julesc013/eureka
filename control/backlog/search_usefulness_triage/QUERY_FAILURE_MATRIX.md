Cross-cutting dimensions:

- external baseline status: all Google and Internet Archive baselines remain `pending_manual_observation`

- Rust parity relevance: Rust should follow stable Python behavior, not lead this usefulness phase
- planner status: Old-Platform Software Planner Pack v0 is now implemented, reducing planner/query-interpretation gaps without claiming new source coverage
