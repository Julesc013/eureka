# Internet Archive Recorded Fixtures Connector

This connector replays a tiny committed Internet-Archive-shaped fixture. It is
recorded fixture coverage only:

- no live Internet Archive API calls
- no scraping
- no crawling
- no network access
- no broad source federation

The fixture exists to exercise source metadata and item-file-list shapes needed
for old-platform-compatible software and later member-level discovery work.
Old-Platform Source Coverage Expansion v0 broadens the committed fixture with
tiny text-safe records for Windows 7 portable apps, Windows XP browser/release
notes, Windows 2000/98 compatibility notes, Mac OS 9 and PowerPC browser
documentation, and Creative CT1740-style driver support media. Those records
are source-shape fixtures only; they are not real redistributed software
binaries and must not be described as a live Internet Archive connector.
