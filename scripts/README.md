# Scripts

This directory holds lightweight repo support scripts.

Current script:

- `demo_resolution_slice.py`: submits and reads the local deterministic gateway thin slice against governed synthetic fixtures, with an optional shared workbench session view-model projection
- `demo_web_workbench.py`: renders the compatibility-first web workbench, deterministic search page, or bundle inspection page either once to stdout, starts a tiny stdlib local server, exports a bounded resolution manifest as JSON, exports a deterministic resolution bundle ZIP to stdout, or inspects a local bundle path as JSON

These scripts are bootstrap utilities, not stable product CLIs.
